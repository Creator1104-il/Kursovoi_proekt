1.Общие сведения

Калькулятор ипотечных вкладов и кредитов — это десктопное приложение, предназначенное для расчета платежей по ипотеке и доходности банковских вкладов. Программа позволяет производить точные финансовые расчеты с учетом различных параметров и сохранять историю выполненных операций.

2.Системные требования

Операционная система: Windows 7/8/10/11

Память: не менее 512 МБ оперативной памяти

Свободное место на диске: 10 МБ

Java Runtime Environment (JRE): версия 8 или выше

3.Установка и запуск

Скачайте файл MortgageCalculatorApp.exe в удобную для вас папку

Дважды щелкните по файлу MortgageCalculatorApp.exe для запуска программы

При первом запуске система может запросить разрешение на запуск приложения - нажмите "Разрешить" или "Да"

4.Интерфейс программы

Программа имеет три основные вкладки:

1. Вкладка "Ипотечный калькулятор"
Расчет ежемесячных платежей по кредиту

Сравнение аннуитетных и дифференцированных платежей

Детальная таблица платежей по месяцам

2. Вкладка "Депозитный калькулятор"
Расчет доходности банковских вкладов

Учет капитализации процентов

Сравнение вкладов с разными условиями

3. Вкладка "История расчетов"
Просмотр всех выполненных расчетов

Сохранение истории в файл

Очистка истории расчетов

5.Работа с ипотечным калькулятором

Ввод данных:
Сумма кредита - введите общую сумму кредита (например: 1000000)

Процентная ставка - укажите годовую процентную ставку (например: 7.5)

Срок - выберите срок кредита в годах (например: 20)

Тип платежей - выберите между аннуитетными и дифференцированными платежами

Валюта - выберите валюту расчетов (RUB, USD, EUR)

Выполнение расчета:
Нажмите кнопку "Рассчитать"

В таблице отобразятся помесячные платежи

В нижней части окна появятся итоговые результаты


Результаты расчета:
Ежемесячный платеж - фиксированная сумма при аннуитетных платежах

Общая сумма выплат - полная стоимость кредита

Переплата по кредиту - сумма выплаченных процентов

График платежей - детализация по каждому месяцу

6.Работа с депозитным калькулятором

Ввод данных:

Сумма вклада - начальная сумма депозита

Процентная ставка - годовая процентная ставка по вкладу

Срок - срок вклада в месяцах

Тип вклада - с капитализацией или без капитализации процентов

Валюта - валюта вклада

Выполнение расчета:
Нажмите кнопку "Рассчитать вклад"

Результаты отобразятся в текстовой области

Результаты расчета:

Итоговая сумма - сумма к получению в конце срока

Начисленные проценты - доход от вклада

Доходность - эффективная процентная ставка

7.История расчетов

Просмотр истории:

Все выполненные расчеты автоматически сохраняются во вкладке "История расчетов"

Для каждого расчета сохраняется дата, тип расчета, параметры и результат

Управление историей:

Очистить историю - удаляет все сохраненные расчеты

Сохранить историю - сохраняет историю в текстовый файл в выбранном месте

Сохранение истории:

Нажмите кнопку "Сохранить историю"

Выберите папку и имя файла для сохранения

Нажмите "Сохранить"

8.Часто задаваемые вопросы

Вопрос: Программа не запускается, что делать?
Ответ: Убедитесь, что на вашем компьютере установлена Java версии 8 или выше. Скачать можно с официального сайта oracle.com

Вопрос: Как сравнить два разных варианта кредита?
Ответ: Выполните расчет для первого варианта, затем измените параметры и выполните расчет снова. Оба расчета сохранятся в истории.

Вопрос: Что такое капитализация процентов?
Ответ: Капитализация - это добавление начисленных процентов к основной сумме вклада, что позволяет в следующем периоде начислять проценты на большую сумму.

Вопрос: Какой тип платежей по кредиту выгоднее?
Ответ: Дифференцированные платежи выгоднее в долгосрочной перспективе, но аннуитетные обеспечивают одинаковый платеж на весь срок кредита.

Вопрос: Можно ли распечатать результаты расчета?
Ответ: Используйте функцию сохранения истории, затем откройте сохраненный файл в текстовом редакторе и распечатайте.
